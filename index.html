WEB의 역사

Internet vs Web
인터넷이 도시라면 웹은 그 도시의 건물이다. 인터넷안에 웹이라는 부분이 존재하고 웹과 동급인 서비스들이 무수히 많다.
1960년에 인터넷이 등장하고 1990년 30년이 지한 후에 웹이 등장한다. 1960년대로 돌아가면 그당시에는 핵이 화두에 있었고 핵공격을 받으면 당시의 통신이 중앙집중적이기 때문에 통신이 마비되는 심각한 상황을 견디기 위해 만들어낸 통신 시스템이 인터넷이다. 인터넷은 중앙이 없고 여러가지 통신장치들이 일종의 전화국과 같은 역할을 수행해 인터넷을 이루게 된다. 30년동안 인터넷은 천천히 확산되지만 이 기간동안 사람들은 인터넷이 세상에 있다는 것조차 잘 몰랐는데 이 이유는 기업이나 연구소, 군대와 같은 거대한 기관에서 사용하는 통신 시스템이 었기 때문이다. 그러다 1990년 웹으 출현으로 완전히 새로운 길을 가게된다.


최초의 웹페이지는 스위스의 유럽인자물리 연구소(CERN)에서 일을 하던 팀 비너스 리라는 사람이 10년간 연구해 1990년 연구실에 인터넷이 들어온 해에 World Wide Web이라는 웹브라우저를 만들게 된다.

서버와 클라이언트

인터넷이 동작하기 위한 본질적으로 2대의 컴퓨터가 필요하다. 하나는 웹 브라우저 하나는 웹서버로 웹브라우저의 주소창에 http://info.cern.ch/index.html 이라는 주소를 입력하게 되면 info.cern.ch라는 컴퓨터에 전기적 신호로 index.html파일을 원한다고 전하게 된다. 그럼 웹서버 프로그램이 하드디스크에 있는 index.html을 전기적 신호로 보내줘 두 컴퓨터가 서로 정보를 주고 받는다.
웹 브라우저는 정보를 요청(request)하고 웹 서버는 응답(respone)하게되는데 이렇게 각각의 역할에 따라 컴퓨터를 부를 때에는 관계로서 고객과 사업자로 요청하는 컴퓨터를 클라이언트(client), 응답하는 컴퓨터를 서버(server)라고 부르기로 한다.

전세계 사용자들에게 내 브라우저를 공유하는 방법

1. 웹 호스팅하기 호스팅 사이트에 내가 만든 html파일을 저장해 웹서버로서의 주소를 활성화 한다.
HTML만을 서비스 한다면 static web hosting이라 부르고 PHP, Python, Ruby같은 언어를 같이 사용한다면 dynamic web hosting이라 한다.
2. 웹 서버를 직접 운영하기
자신의 컴퓨터에 웹 서버 프로그램을 설치한다. 무료로 이용할 수 있는 제품군에는 Apache가 있는데 window macOS Linux 세가지 운영체제별로 설치 할 수 있다. 아파치를 이용해 서버를 만들어 웹페이지를 운영하면 처음 주소값은 자신의 ip주소를 이용해 사이트 주소를 만들게 된다.
웹 서버를 이용하는 경우 주소에는 http://(HyperText Transfer Protocol)즉 웹브라우저와 웹서버가 서로 통신할 때 사용되는 규약을 이용해 데이터를 가져오고, 컴퓨터 내의 파일을 불러오는 경우는 주소창에 file://로 뜨게된다.
